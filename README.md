**EvilEye** is a powerful, upgraded alternative to [CamPhish.](https://github.com/techchipnet/CamPhish) It allows users to generate deceptive links that capture images from the target's front camera upon permission. Unlike the original, EvilEye includes modern social engineering templates designed for 2025-2026 trends.


## 🚀 Key Features
While original tools are limited, **EvilEye** provides 5 high-conversion templates:

-  **📺 YouTube TV:** A realistic youtube video interface.
-  **🎂 Birthday Wish:** A personalized greeting card template.
-  **📈 Fake Insta Follower:** A "Get Free Followers" dashboard to entice clicks.
-  **📩 Secret Message:** An anonymous messaging portal (perfect for "NGL" style baits).
-  **🤖 Fake AI:** A futuristic AI-chat interface.

## 🛠️ Installation

### Linux

#### 🔹Clone the repo in your system.
```
git clone https://github.com/rgsecteam/evel-eye.git
```
#### 🔹Go to the **evil-eye** directory
```
cd evel-eye
```
#### 🔹Give this main file **evil.sh** executeable permission.
```
chmod +x evil.sh
```
#### 🔹Run this file
```
./evil.sh
```
